hyper.chrome
============

Hyper client for chrome

## Install

1. [Download the extension](https://github.com/hypergroup/hyper.chrome/archive/master.zip)
2. Unpack the zip file
3. Open the Chrome extensions tab]. Go to chrome://extensions/
4. Click the Developer mode check box
5. Click the Load Unpacked Extension button
6. Navigate to the unzipped folder and click the Select button
